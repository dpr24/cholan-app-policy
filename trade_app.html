I. Entrance Gateway (Before Trade)

Hard Chronological Rules

if (Time < 09:30 IST) → Trade Blocked
if (OpenPositions ≥ 1) → Trade Blocked
if (ExecutedTradesToday ≥ 1) → Trade Blocked
if (CurrentTrendTradeCount ≥ 3) → Trade Blocked

Confluence Engine Validation

  • TREND_DIRECTION must be VALID
  • MARKET_STRUCTURE must be VALID
  • KEY_LEVELS must be VALID
  • Grid Density ≥ 70%
  • ENTRY_PATTERN must be VALID on 15M or 5M timeframe

II. Contract & Order Selection

Instrument Choosing

if (MarketDirection == UP)
    Select CALL Option (CE)

if (MarketDirection == DOWN)
    Select PUT Option (PE)

Strike Selection

CE Strike = floor(SpotPrice - 150)

PE Strike = ceil(SpotPrice + 150)

Strike must be ≥150 points ITM

Expiry & Premium Logic

  • Target expiry: ~21 trading days
  • Select contract with lower premium
  • Reject if premium ≥ highest premium of last 3 days

III. Order Execution System

Pre-Order Validation

if (Quantity < 1 OR Quantity > 10000)
    REJECT

if (Price <= 0)
    REJECT

PriceDiff = abs(Price - LTP) / LTP * 100

if (PriceDiff > 3%)
    REJECT

Reward = abs(Target - Price)
Risk = abs(Price - SL)

if (Reward / Risk < 2)
    REJECT

Position Sizing

Qty = floor( TradeCapital / (Premium * 25) )

FinalQty = floor(Qty / 50) * 50

if (FinalQty < 50)
    REJECT

Broker Parameters

  • Product Type: MIS
  • Order Type: LIMIT
  • Validity: DAY
  • Entry: BUY
  • Exit: SELL

IV. Position Monitoring

Stop Loss / Target Updates

if (NewSL > 0 AND NewSL < EntryPrice)
    PERMIT

if (NewTarget > 0 AND NewTarget > EntryPrice)
    PERMIT

Time Based Holding

if (CurrentTime - EntryTime >= 120 minutes)
    EXIT

News Based Adjustment

if (NewsRestrictionStatus == RESTRICTED)

Block new trades

Reduce position to 33%

floor(CurrentQty * 0.33)

V. Exit Conditions

  • Stop Loss triggered
  • Target reached
  • Trade duration reaches 2 hours
  • News override
  • Manual dashboard exit

VI. Mathematical Logic Specification

MACD Calculation

Multiplier = 2 / (Period + 1)

EMA_today =
(Price_today - EMA_yesterday) * Multiplier
+ EMA_yesterday

MACD Line = EMA(12) - EMA(26)

Signal Line = EMA(MACD, 9)

Histogram = MACD - Signal

Retracement

Range = HighestHigh - LowestLow

Retracement % =
((HighestHigh - CurrentClose) / Range) * 100

Zig-Zag Validation

Pullback 1 → ≥ 70%

Pullback 2 → ≥ 60%

Pullback 3 → ≥ 55%
